# KAIROS — AI Decision Intelligence for Collections

This repository powers [runkairos.com](https://runkairos.com), the home of KAIROS: a GCC-native collections decision intelligence platform for banks, Islamic banks, digital banks, and BNPL providers.

KAIROS scores every delinquent account in real time and recommends the next-best action — channel, timing, tone, and treatment — so institutions recover more, earlier in the DPD cycle, at a lower cost to collect.

## Repository notes

The site is a self-contained static build deployed via Cloudflare Pages. `_headers` defines HTTP response headers, `robots.txt` and `sitemap.xml` handle crawling, and `html2pdf.bundle.min.js` supports in-browser document export.
